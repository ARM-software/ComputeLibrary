44 /** Basic function to execute GEMMLowpMatrixMultiplyCore on OpenCL. This function calls the following OpenCL kernels:
45  *
46  * -# @ref CLGEMMReshapeRHSMatrixKernel (if the output tensor is a matrix)
47  * -# @ref CLGEMMLowpMatrixMultiplyKernel (if the parameter "reshape_b_only_on_first_run" of GEMMInfo is FALSE)
48  * -# @ref CLGEMMLowpMatrixMultiplyReshapedOnlyRHSKernel (if the parameter "reshape_b_only_on_first_run" of GEMMInfo is TRUE)
49  * -# @ref CLGEMMLowpMatrixAReductionKernel (if the offset of matrix B is not 0)
50  * -# @ref CLGEMMLowpMatrixBReductionKernel (if the offset of matrix A is not 0)
51  * -# @ref CLGEMMLowpOffsetContributionKernel (if gemm_info.gemmlowp_output_stage == NONE)
52  * -# @ref CLGEMMLowpOffsetContributionOutputStageKernel (if gemm_info.gemmlowp_output_stage != NONE)

78  *
79  * @param[in] a First input tensor (Matrix A). Data type supported: QASYMM8/QASYMM8_SIGNED.

81  * @param[in] c Third input tensor (Matrix C). It can be a nullptr. Data type supported: S32

83  * @param[in] gemm_info (Optional) Specifies if the matrix A and/or matrix B have been reshaped and
84  * if the reshape of matrix B should be executed only for the first run
