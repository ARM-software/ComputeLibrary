39 /** Basic function to compute a SoftmaxLayer.
40  *
41  * Softmax is calculated by :
42  * @f[ out = exp((x - max(x)) * beta) / sum(exp((x - max(x)) * beta)) @f]
43  *
44  * Log Softmax is calculated by :
45  * @f[ out = (x - max(x) * beta) - log(\sum{e^{x - max(x) * beta}}) @f]
46  */
