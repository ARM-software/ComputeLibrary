60  /** Configure operator for a given list of arguments
61  *
62  * Valid data layouts:
63  * - All
64  *
65  * Valid data type configurations:
66  * |a |b |c |d |
67  * |:------------|:-----------|:---------|:--------------|
68  * |F32 |F32 |F32 |F32 |
69  * |F16 |F16 |F16 |F16 |
70  * |BFLOAT16 |BFLOAT16 |BFLOAT16 |FP32 |
71  *
72  * @note GEMM: General Matrix Multiply - [alpha * A * B + beta * C].
73  * @note GEMM: The tensors a, b, c, d must have the same data type. You should not mix data types when calling this function.
74  *
75  * @note Batched GEMM only supports broadcasting cases where RHS rank < LHS rank but not the other way around
76  *
77  * @param[in] a First input tensor info (Matrix A or Vector A). Data type supported: BFLOAT16/F16/F32
78  * @param[in] b Second input tensor info (Matrix B). Data type supported: same as @p a
79  * @param[in] c Third input tensor info (Matrix C). It can be a nullptr if just the multiplication between @p a and @p b is needed. Data type supported: same as @p a
80  * @param[out] d Output tensor info. Data type supported: same as @p a
81  * @param[in] alpha Weight of the matrix product
82  * @param[in] beta Weight of matrix C
83  * @param[in, out] gemm_info (Optional) Specifies if the matrix A and/or matrix B have been reshaped and
84  * if the reshape of matrix B should happen only for the first run
85  */
86  void configure(const ITensorInfo *a,
87  const ITensorInfo *b,
88  const ITensorInfo *c,
89  ITensorInfo *d,
90  float alpha,
91  float beta,
92  const GEMMInfo &gemm_info = GEMMInfo());
