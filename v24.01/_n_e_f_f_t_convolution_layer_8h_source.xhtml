45 /** Basic function to execute FFT-based convolution on CPU. This function calls the following functions/kernels:
46  *
47  * -# @ref NEPermute Permute input if NHWC(only NCHW is supported).
48  * -# @ref NEPadLayer Pad input.
49  * -# @ref NEFFT2D Forward transform to the frequency domain.
50  * -# @ref NEComplexPixelWiseMultiplication Complex element-wise product of input and the weights.
51  * -# @ref NEReductionOperation Reduction across channels.
52  * -# @ref NEFFT2D Inverse transform back to the time domain.
53  * -# @ref NEStridedSlice Extract valid output.
54  * -# @ref NEArithmeticAddition Add bias.
55  * -# @ref NEActivationLayer Perform activation.
56  * -# @ref NEPermute Permute output if NHWC(only NCHW is supported).
57  */
