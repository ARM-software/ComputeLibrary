Parameters
[in]compile_contextThe compile context to be used.
[in]srcSource input tensor info. Can be nullptr when using "Add" Scatter Function with zero initialization.
[in]updatesTensor info for tensor storing update values to use for scatter function. Data types supported: same as src.
[in]indicesTensor info for tensor storing indices to use for scatter function. Data types supported: U32 only.
[out]dstOutput tensor to store the result of the Scatter Function. Data types supported: same as src and updates.
[in]Scatter_infoContains Scatter operation information described in ScatterInfo.
