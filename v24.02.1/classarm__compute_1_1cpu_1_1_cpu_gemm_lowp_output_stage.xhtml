Detailed Description

Basic function to execute GEMMLowpQuantizeDown kernels.

This function calls the following kernels:

  1. kernels::CpuGemmLowpQuantizeDownInt32ScaleKernel
  2. kernels::CpuGemmLowpQuantizeDownInt32ToUint8ScaleByFixedPointKernel
  3. kernels::CpuGemmLowpQuantizeDownInt32ToInt8ScaleByFixedPointKernel
  4. kernels::CpuGemmLowpQuantizeDownInt32ToInt16ScaleByFixedPointKernel
