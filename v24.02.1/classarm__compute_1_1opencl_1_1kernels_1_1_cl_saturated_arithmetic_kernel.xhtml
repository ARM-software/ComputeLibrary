Static function to check if given info will lead to a valid configuration of ClSaturatedArithmeticKernel.

Arithmetic operations with saturation.

Parameters
[in]compile_contextThe compile context to be used.
[in]opArithmetic operation to be executed.
[in]input1First tensor input info. Data types supported: U8/QASYMM8/QASYMM8_SIGNED/S16/QSYMM16/F16/S32/F32.
[in]input2Second tensor input info. Data types supported: Same as input1.
[in]outputOutput tensor info. Data types supported: Same as input1.
[in]policyPolicy to use to handle overflow.
[in]act_info(Optional) Activation layer information in case of a fused activation.
