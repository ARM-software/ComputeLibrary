Detailed Description

Basic function to execute GEMMLowpQuantizeDown kernels.

This function calls the following kernels:

  1. NEGEMMLowpQuantizeDownInt32ScaleKernel
  2. NEGEMMLowpQuantizeDownInt32ToUint8ScaleByFixedPointKernel
  3. NEGEMMLowpQuantizeDownInt32ToInt8ScaleByFixedPointKernel
  4. NEGEMMLowpQuantizeDownInt32ToInt16ScaleByFixedPointKernel
