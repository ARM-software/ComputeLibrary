This performs an activation function floating point inputs.

Note
Datatype should be given as a preprocessor argument using -DDATA_TYPE=type. e.g. -DDATA_TYPE=short
Activation function should be given as a preprocessor argument using -DNAME. e.g. -DTANH
Distinction between floating point and integer is done using -DTYPE_FP and -DTYPE_INT preprocessor argument
A, B variables required by some activation functions are set using -DA= and -DB= respectively.
