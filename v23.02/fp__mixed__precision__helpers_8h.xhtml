MIXED_PRECISION_ELTWISE_OP_BLOCK_BROADCAST

Mixed-Precision-Aware Elementwise Op Broadcast Block Performs OPERAND1 = OP(OPERAND1, OPERAND2)

Note
Only support: case 1 broadcast in Y dimension : Operand1 [YxX] + Operand2 [1xX]; this means N0 > 1 case 2 broadcast in both Y and X dimensions : Operand1 [YxX] + Operand2 [1x1] (scalar) ; this means N0 == 1 Does NOT support broad cast in X dimension: Operand1 [YxX] + Operand2 [Yx1]; this means M0 should never == 1
Parameters
[in]OPThe elementwise post op
[in]M0The number of consecutive rows, > 1
[in]N0The number of consecutive columns, >= 1
[in]OPERAND1The basename of the first and result operand variables
[in]OPERAND2The basename of the second operand variables
[in]DATA_TYPE_ACCUMULATRHigher-precision accumulator data type in case of mixed-precision op
[in]CONVERTED_OPERAND2The basename of the second operand variables converted to higher-precision in case of mixed-precision op
#define MIXED_PRECISION_ELTWISE_OP_BLOCK_BROADCAST(OP, M0, N0, OPERAND1, OPERAND2, DATA_TYPE_ACCUMULATOR, CONVERTED_OPERAND2)   ELTWISE_OP_BLOCK_BROADCAST(OP, M0, OPERAND1, OPERAND2##0);
