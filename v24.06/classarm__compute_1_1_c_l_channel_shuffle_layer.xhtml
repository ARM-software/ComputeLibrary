Detailed Description

Basic function to run CLChannelShuffleLayerKernel.

Note
The function performs a channel shuffle operation on the input tensor. Given NCHW tensor with group G, it will first divide the channels into G groups, C = (G * C'), and perform a transpose of the channel, which gives C = (C' * G). for more details see: https://arxiv.org/pdf/1707.01083.pdf
