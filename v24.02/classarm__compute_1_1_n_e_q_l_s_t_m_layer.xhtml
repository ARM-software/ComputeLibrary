Detailed Description

Basic function to run NEQLSTMLayer.

This function calls the following kernels:

  1. NEActivationLayer Activation functions (tanh and logistic)
  2. NEArithmeticAddition Elementwise addition
  3. NEArithmeticSubtraction Elementwise subtraction
  4. NECopy Copy kernel for copying output_state_out to output
  5. NEGEMMLowpMatrixMultiplyCore Quantized matrix multiplication core. Accumulators are 32-bit integers
  6. NEGEMMLowpOutputStage Convert 32-bit integers into QSYMM16
  7. cpu::kernels::CpuGemmLowpMatrixAReductionKernel For precomputing effective biases to use
  8. NEPixelWiseMultiplication Elementwise multiplication
  9. NETranspose Transpose function for reshaping the weights
