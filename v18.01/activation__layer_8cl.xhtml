This performs an activation function floating point inputs.

Note
In order to perform the activation function "in-place", the pre-processor -DIN_PLACE must be passed at compile time
Datatype should be given as a preprocessor argument using -DDATA_TYPE=type. e.g. -DDATA_TYPE=short
Vector size should be given as a preprocessor argument using -DVEC_SIZE=size. e.g. -DVEC_SIZE=16
Activation function should be given as a preprocessor argument using -DACT=name. e.g. -DACT=TANH
A, B variables required by some activation functions are set using -DA_VAL= and -DB_VAL= respectively.
In case of fixed point calculations the fixed point position is passed using -DFIXED_POINT_POSITION=position. e.g. -DFIXED_POINT_POSITION=3.
