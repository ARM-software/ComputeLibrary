46  /** Set the src, weights, and dst tensors.
47  *
48  * @note: DirectConvolution only works in the following configurations:
49  * 1x1 convolution with stride_x = 1/2/3, stride_y = 1/2/3
50  * 3x3 convolution with stride_x = 1/2/3, stride_y = 1/2/3
51  *
52  * @param[in] src The input tensor to convolve. 3 lower dimensions represent a single input [width, height, IFM],
53  * while every optional dimension from 4 and above represent a batch of inputs. Data types supported: F16/F32.
54  * @param[in] weights Weights tensor. Weights are 4D tensor with dimensions [kernel_x, kernel_y, IFM, OFM].
55  * The 3rd dimension must be the same as the input's volume 3rd dimension.
56  * Data type supported:Same as @p input.
57  * @param[out] dst Output tensor.
58  * The 3rd dimensions must be equal to the 4th dimension of the @p kernels tensor. Data types supported: F16/F32
59  * @param[in] conv_info Contains padding and stride information described in @ref PadStrideInfo.
60  */
