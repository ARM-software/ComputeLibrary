Initialize the kernel's input and output.

This kernel performs matrix multiplication of lhs and rhs:

dst = matmul(lhs, rhs)

Valid data layouts:

  • All

Valid data type configurations:

lhs rhs dst
F32 F32 F32
F16 F16 F16

Shape definitions: Dim0, Dim1, Dim2... lhs: [ K, M, Batch dims...] rhs: [ N, K, Batch dims...] dst: [ N, M, Batch dims...]

Valid shape configurations:

  • K must be a multiple of 4 (MMUL_K0).
  • No broadcasting in batch dimensions. I.e. batch dims must be the same across lhs, rhs and dst
Parameters
[in]compile_contextThe compile context to be used.
[in]lhsInput tensor info for the LHS matrix.
[in]rhsInput tensor info for the RHS matrix.
[in]biasBias tensor info. Can be nullptr. Data type supported: Same as lhs.
[out]dstOutput tensor info.
[in]matmul_infoAttributes for Batch MatMul Kernel
