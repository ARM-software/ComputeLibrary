Detailed Description

GEMM reshape information class.

This class stores the necessary information about matrix A and matrix B reshape.

The matrix A can only be reshaped through CLGEMMInterleave4x4Kernel or NEGEMMInterleave4x4Kernel or GCGEMMInterleave4x4Kernel Note: Optionally just for CLGEMMInterleave4x4Kernel is it possible to set mult_interleave4x4_height, the multiplication factor for the height of the 4x4 interleaved block

The matrix B can only be reshaped through CLGEMMTranspose1xWKernel or NEGEMMTranspose1xWKernel or GCGEMMTranspose1xWKernel Note: Optionally just for CLGEMMTranspose1xWKernel is it possible to set mult_transpose1xW_width, the multiplication factor for the width of the 1xW transposed block
