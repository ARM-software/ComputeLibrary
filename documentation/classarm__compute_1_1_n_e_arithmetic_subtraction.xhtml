Detailed Description

Basic function to run NEArithmeticSubtractionKernel.

Note
The tensor data type for the inputs must be U8/QASYMM8/S16/F16/F32.
The function performs an arithmetic subtraction between two tensors.

This function calls the following kernels:

  1. NEFillBorderKernel (In case of broadcasting, in the input being broadcasted)
  2. NEArithmeticSubtractionKernel
