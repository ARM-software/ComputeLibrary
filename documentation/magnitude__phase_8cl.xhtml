Calculate the magnitude and phase of given the gradients of an image.

Note
Magnitude calculation supported: L1 normalization(type = 1) and L2 normalization(type = 2).
Phase calculation supported: Unsigned(type = 1) [0,128] and Signed(type = 2) [0,256).
Attention
To enable phase calculation -DPHASE="phase_calculation_type_id" must be provided at compile time. eg -DPHASE=1
To enable magnitude calculation -DMAGNITUDE="magnitude_calculation_type_id" must be provided at compile time. eg -DMAGNITUDE=1
Datatype of the two inputs is passed at compile time using -DDATA_TYPE. e.g -DDATA_TYPE=short. Supported data_types are: short and int
