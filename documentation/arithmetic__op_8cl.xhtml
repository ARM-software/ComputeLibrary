This function add two images.

Attention
The input and output data_types need to be passed at compile time using -DDATA_TYPE_IN1, -DDATA_TYPE_IN2 and -DDATA_TYPE_OUT: e.g. -DDATA_TYPE_IN1=uchar -DDATA_TYPE_IN2=uchar -DDATA_TYPE_OUT=short
To perform saturating operation -DSATURATE has to be passed to the compiler otherwise wrapping policy will be used.
