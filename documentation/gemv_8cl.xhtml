This kernel applies dot product to each plane on the input tensor and the corrispective column of the reshaped weight tensor.

Note
Datatype and source width and height should be given as a preprocessor argument using -DDATA_TYPE=type, -DSRC_WIDTH=width and -DSRC_HEIGHT=height. e.g. -DDATA_TYPE=short
