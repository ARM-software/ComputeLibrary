Initialise the kernel's inputs, output and convertion policy.

Parameters
[in]input1An input tensor. Data types supported: U8/QS8/QS16/S16/F16/F32.
[in]input2An input tensor. Data types supported: same as input1.
[out]outputThe output tensor, Data types supported: same as input1. Note: U8 (QS8, QS16) requires both inputs to be U8 (QS8, QS16).
[in]scaleScale to apply after multiplication. Scale must be positive and its value must be either 1/255 or 1/2^n where n is between 0 and 15. For QS8 and QS16 scale must be 1.
[in]overflow_policyOverflow policy. Supported overflow policies: Wrap, Saturate
[in]rounding_policyRounding policy. Supported rounding modes: to zero, to nearest even.
