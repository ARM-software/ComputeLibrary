Initialise the kernel's input, output and border mode.

Valid configurations (Input1,Input2) -> Output :

  • (U8,U8) -> U8
  • (U8,U8) -> S16
  • (S16,U8) -> S16
  • (U8,S16) -> S16
  • (S16,S16) -> S16
  • (F16,F16) -> F16
  • (F32,F32) -> F32
  • (QASYMM8,QASYMM8) -> QASYMM8
  • (QSYMM16,QSYMM16) -> QSYMM16

[in]input1An input tensor. Data types supported: U8/QASYMM8/S16/QSYMM16/F16/F32
[in]input2An input tensor. Data types supported: U8/QASYMM8/S16/QSYMM16/F16/F32
[out]outputThe output tensor. Data types supported: U8/QASYMM8/S16/QSYMM16/F16/F32.
[in]policyOverflow policy.

Warning
