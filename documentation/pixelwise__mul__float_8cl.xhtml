Performs a pixelwise multiplication with float scale of either integer or float inputs.

Attention
The inputs and output data types need to be passed at compile time using -DDATA_TYPE_IN1, -DDATA_TYPE_IN2 and -DDATA_TYPE_OUT: e.g. -DDATA_TYPE_IN1=uchar -DDATA_TYPE_IN2=ushort -DDATA_TYPE_OUT=short
The data type of the intermediate result of the multiplication should passed as well using -DDATA_TYPE_RES. e.g. If one of inputs is S16 -DDATA_TYPE_RES=int should be passed else -DDATA_TYPE_RES=short.
-DDATA_TYPE_FLOAT must be passed if floating point inputs are provided.
